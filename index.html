    MailAutomata Project

    MailAutomata is an advanced software solution designed to automate the sorting, information extraction, and distribution of documents. The system enhances efficiency and accuracy in handling repetitive tasks, ensuring that documents are processed and sent to the appropriate recipients with minimal manual intervention.

    Features:

    Document Sorting and Information Extraction:

    • Sorting by Name: Automatically scans and sorts documents based on their names, ensuring organized handling and easy retrieval.
    • VAT Number Isolation: Extracts VAT numbers from invoices to ensure accurate record-keeping and compliance with financial regulations.

    Automated Document Distribution:

    • Invoice Handling: Identifies invoices and sends them to a predefined list of recipients.
    • Delivery Note Handling: Recognizes delivery notes and routes them to a different set of recipients.
    • Recipient Management: Allows users to add, modify, or delete recipient information in the database for up-to-date contact management.

    Comprehensive Database Management:

    • Recipient Database: Stores all relevant data of recipients, facilitating quick access and efficient management.
    • Document Tracking: Records detailed information about each document, including sending details and timestamps.
    • Data Extraction: Enables users to export document and sending details into an Excel table for analysis and reporting.

    User-Friendly Control Panel:

    • Program Setup: Provides options to configure program runtime, archive directory, and other settings.
    • Manual Shutdown: Allows users to manually shut down the program when necessary.

    Technologies Used:

    • Win32 API and C++: The core functionalities of MailAutomata are built using the Win32 API and C++, ensuring high performance and reliability.
    • Database Integration: Efficient data management using SQL queries and the most commonly used database with the Win32 API.
    • MFC (Microsoft Foundation Classes): The application interface is designed using MFC for a seamless and intuitive user experience.

    Purpose:

    MailAutomata is designed to automate repetitive document handling tasks, perform them faster, and eliminate human error. By leveraging advanced technologies and intelligent automation, MailAutomata ensures that your document processing and distribution are efficient, accurate, and hassle-free.

    Conclusion:

    MailAutomata is your solution to automating the tedious and error-prone task of document sorting and distribution. With its intelligent features, robust technology stack, and user-friendly interface, MailAutomata stands at the forefront of innovation, pushing the boundaries of what automation can achieve. Embrace the future of automated document management with MailAutomata.

    Bela Score Calculator

    The Bela Score Calculator is an Android application designed to simplify scorekeeping for the card game Bela. The app keeps track of the game score, calls, and displays the results in an easy-to-read grid format. It also saves scores and games in a database for future reference.

    Features:

    Score Management:

    • Base Game Score: Automatically divides the base score of 162 points between two teams, ensuring accurate scorekeeping.
    • Add Calls: Allows players to add calls and integrates them into the overall score seamlessly.
    • Score Display: Presents the scores in a clear and organized grid format, making it easy to follow the game progress.

    Game Settings and Customization:

    • Win Condition: Sets a win condition based on a predefined score threshold, declaring the winner when one team surpasses it.
    • Team Names: Provides the ability to change team names, adding a personal touch to the game.

    Data Persistence:

    • Score and Game Saving: Saves scores and game data in a database, allowing players to resume games or review past results.

    Technologies Used:

    • C# and .NET MAUI: The application is developed using C# and .NET MAUI, offering a robust and scalable framework for building Android applications.
    • SQLite Database: Utilizes SQLite for efficient data storage and management, ensuring reliable data persistence and retrieval.

    Purpose:

    The Bela Score Calculator is designed to provide an easy and accurate way to keep score in the card game Bela. By automating score calculations and providing a user-friendly interface, the app simplifies the game for players, allowing them to focus on the fun rather than the math.

    Conclusion:

    The Bela Score Calculator is your go-to solution for hassle-free scorekeeping in the card game Bela. With its intuitive features, solid technology stack, and user-centric design, it ensures that your game sessions are enjoyable and stress-free. Embrace the convenience and accuracy of the Bela Score Calculator and enhance your Bela playing experience.

    CV recursion

    This web-based CV is designed to present professional information, project showcases, and contact details in an engaging and interactive format. The webpage is structured to provide a clear and comprehensive overview of the individual's qualifications, experiences, and projects.

    Features:

    1. Introduction Section:

    • Overview: Provides a brief introduction to the individual, highlighting key professional attributes and career goals.
    • Personal Information: Displays essential details such as name, date of birth, place of birth, contact information, and languages spoken.

    2. CV Section:

    • Experience: Lists professional experiences, including job titles, roles, and responsibilities, arranged in a visually appealing and organized manner.
    • Skills: Highlights specific skills, both technical and general, with visual indicators of proficiency levels.
    • Driving Categories: Displays driving categories (e.g., B, AM, A) in an interesting and clear format.

    3. Portfolio Section:

    • Project Icons: Displays various project icons in a visually appealing grid layout.
    • Interactive Modals: Clicking on any project icon opens a detailed modal view with images and descriptions of the project, providing an in-depth look at the work.

    4. Contact Section:

    • Contact Form: Allows visitors to send messages directly through the website. The form includes fields for name, email, subject, and message.
    • Social Media Links: Includes icons linking to social media profiles for additional contact options.

    Technologies Used:

    • HTML5: For the structure and content of the webpage.
    • CSS3: For styling, layout, and responsive design to ensure the page looks good on all devices.
    • JavaScript: For interactivity, including opening and closing modals and handling form submissions.

    Purpose:

    The web CV serves as an online portfolio and resume, offering a modern and interactive way for potential employers and collaborators to explore the individual's professional background, skills, and projects. It combines clear presentation with engaging functionality, making it easy to navigate and visually appealing.
